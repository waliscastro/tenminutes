Photo Feed is an easy solution to display your Instagram Feed within website.

== Description ==

**Instagram media showcase plugin**

Display your Instagram account feed within website as different layouts. Very easy to setup and will be ready within few seconds. Just install and check how it works. Also you can check the plugin website for working preview. it's compatible with IE 10+, Safari 6.1+ and all other modern web browsers.

**How to use:** install and activate like any other plugin, Then go to *Dashboard > Settings > Photo Feed* and connect your Instagram account. After successful connection update the necessary display settings and copy, paste the visible shortcode **[photo-feed]** within the pages, posts, widgets etc.

**Demo:** check [Photo Feed demo](https://photofeed.ga/) or [Screenshots](#screenshots) section for preview.

*Photo Feed plugin is under active development and more features will be added with time.*

= How to Install =

You can install in the same way you had installed any other plugin OR check [instructions here](https://wordpress.org/support/article/managing-plugins/)


== Frequently Asked Questions ==

= Why pictures are not showing in the page? =

make sure your Instagram account is public and successfully connected to the website. OR try to disconnect and connect again (if already connected).

= Error: No working transports found =

please check if server or application configured correctly and necessary extensions are active on the server.
